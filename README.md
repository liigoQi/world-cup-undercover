# world-cup-undercover

模拟微信群聊的交互式网页游戏。被拉进一个世界杯观赛群，在12个性格鲜明的群友中卧底潜伏，通过聊天互动学习越位规则、世界杯常识和群聊生存术，最终获得"懂球浓度"评分。纯前端单文件，零依赖，手机端适配。

## 在线体验

直接浏览器打开 `index.html` 即可。

## 玩法

- 设置群昵称加入观赛群
- 跟随群聊推进剧情，在关键时刻选择发言内容
- 第四章包含越位规则互动教学小游戏
- 第六章在4个场景中模拟群聊发言，学习不暴露新手身份
- 终章经历20+条消息的高压群聊测试
- 获得"懂球浓度"百分比和身份等级（Lv.1 路人 → Lv.10 世界杯传教士）

## 技术

纯 HTML/CSS/JS 单文件，无任何依赖。微信聊天界面风格，移动端触摸适配。
